Open project for all of US, named Base.


1. Select by Attributes...
2. "STATE_FIPS" NOT LIKE '02' AND "STATE_FIPS" NOT LIKE '15'
3. Editor | Start Editing...
4. Right click Records | Delete Selected
5. Editor | Save Edits
6. Editor | Stop Editing
